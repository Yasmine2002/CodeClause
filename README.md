# CodeClause
Time Conversion Application:
----------------------------
This is a simple Java command-line application that allows users to convert the current time from one time zone to another. The application takes the source country and target country time zone IDs as input and displays the converted time in the target country's time zone.

Features:
---------
Converts the current time from one time zone to another.
Supports a wide range of time zones.
